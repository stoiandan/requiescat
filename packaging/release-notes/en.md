This release adds GPS coordinates for graves, so each grave can store and display an exact location alongside its map position. It also adds delimiter drawing tools for cemetery walls and walking paths, with selectable wall/road styles, shared drawing colors, move/erase support, post-creation rotation for graves and delimiters, persistence, and PDF export. It also simplifies the cemetery library menu, refreshes Romanian copy, and includes updater and internal refactors that keep the app easier to maintain.
